# PdfAsk

使用pdfjs解析PDF，将PDF内容向量化存储。对用户提问和PDF内容向量化对比之后，通过gpt3.5 对话模式回答用户的提问。

## 功能
- 导入PDF并解析 (done)
- 根据PDF内容回答提问 (done)
-  提供回答内容所在PDF的位置 (TODO)
-  回答同提问的语言 (TODO)
-  支持导出对话内容 (TODO)
-  提供分享链接 (TODO)
-  支持客户端使用(Tauri/Electron) (TODO)
